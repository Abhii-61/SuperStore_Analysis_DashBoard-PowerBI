# SuperStore_Analysis_DashBoard-PowerBI
Dashboard to analyse a SuperStore Sales using PowerBI

 Super Store Sales Analysis 
Dashboard 
1. Introduction 
The Super Store Sales Analysis project is a business intelligence dashboard 
created using Power BI. 
The objective of this project is to analyze sales, profit, customer segments, ship 
modes, and product sub-categories to provide meaningful insights that can 
support data-driven decision-making. 
This dashboard helps stakeholders quickly understand overall performance 
trends, identify high-profit areas, analyze seasonal patterns, and compare the 
contribution of different product and customer segments. 
2. Project Objectives 
The primary goals of this project were: 
1. To design an interactive dashboard that provides a 360-degree view of 
Super Store sales performance. 
2. To evaluate key metrics such as: 
● Total Sales 
● Total Profit 
● Ship Modes 
● Product Sub-Categories 
3. To analyze sales distribution across: 
● Customer Segments 
● Monthly Sales Trends 
● Ship Mode Performance 
● Sub-Category Contribution 
4. To allow business users to apply dynamic filters for better exploratory 
analysis. 
3. Tools & Technologies Used 
● Power BI Desktop 
● Data Cleaning and Modeling (Power Query) 
● Data Visualization 
● DAX Measures for advanced calculations
